## SmartBatteryV1-1.1.6

**日期：** 2026-03-26  

**摘要：** MQTT 遠端馬達控制（對齊二代 `motorControl` 指令）。

**變更：**

- `mqtthelper`：訂閱並解析 `/motorControl`，發射 `motorControlReceived`；payload 欄位 `action`：`forward`／`back`／`stop` 對應 `DeviceAPI_MotorForward`／`MotorBack`／`MotorStop`。
- `main.cpp`：收到後暫停 idle、停止首頁輪播、設定維護／暫停狀態與 `keepStop`、切至錯誤／維護頁（`err_dlg`）、執行馬達動作並 `checkStatus`。一代以 `LedAPI_Select(2)` 對應二代之綠燈指示（二代原始碼為 `LedAPI_Set(0x00ff00)`）。

## SmartBatteryV1-1.1.5

**日期：** 2026-03-26  

**摘要：** MQTT 遠端電池重量歸零（對齊二代 `weightReset` 指令）。

## 維護方式

1. 修改程式前，先於 `global.cpp` 更新 `SWVersion`（見 [版本與提交習慣.txt](版本與提交習慣.txt)）。
2. 於本檔最上方新增一節 `## SmartBatteryV1-x.y.z`，填日期與變更摘要（必要時列檔案或模組）。
3. 若為正式發佈，於對應 commit 再打同名 Git tag 並 `git push origin <tag>`。
